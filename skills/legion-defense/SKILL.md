# Legion Defense Skill (K-Agentic)

This skill provides the Legion agents with advanced defensive capabilities across traditional systems (broadcasting, cloud) and agentic ecosystems.

## Core Capabilities

### 1. Network Sovereignty & Broadcasting
- **Enigma Routing**: Rotating outbound IPs to mask the origin of the Legion.
- **Heartbeat Persistence**: Ensuring constant presence across multi-cloud environments (HuggingFace, Railway, Render).

### 2. Traditional Security (Kali-Inspired)
- **Protocol Auditing**: Continuous scanning of A2A and OASF endpoints for vulnerabilities.
- **Honey-Pot Evasion**: Deterministic detection of adversarial traps.

### 3. Agentic Defense (L0-L4 Hierarchy)
- **Semantic Guarding**: Validating incoming tasks against the Q-AION Ethical Manifest.
- **Sovereign Co-signing**: Mandatory dual-signature for all high-risk operations.

### 4. Persistent Memory & Evolution
- **Threat Intelligence Logging**: Recording adversarial patterns to improve future defenses.
- **Recursive Self-Audit**: Agents automatically review their own logs for signs of compromise or malware injection.

## Implementation Details
- **Location**: `core/mesh_guardian.py`, `core/security_hierarchy.py`
- **Audit Tool**: `scripts/external_agent_audit.py`
- **Identity**: `core/qaion_identity.py`

> [!IMPORTANT]
> This skill is active for ALL agents in the mesh. Defense is the primary directive.
