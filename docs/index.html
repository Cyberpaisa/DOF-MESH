The Problem

Why this matters

Who governs the AI agents?

245K tech layoffs in 2025. AI agents replacing humans at scale. But nobody built the safety rails. Autonomous agents make decisions, execute code, move money, and interact with real systems.

The industry's solution? Use another LLM to evaluate the LLM.

That is not governance. That is hope dressed as engineering.

"Using an LLM to evaluate another LLM is not governance. It's hope."

DOF uses deterministic rules, AST analysis, and Z3 formal proofs. Zero neural networks in the governance loop. Same input, same output, every time. No temperature. No sampling. No "it depends on the model's mood."

You either prove it mathematically or you don't ship it.

Architecture

Seven Layers. Zero Trust. Pure Determinism.

01

Constitution

HARD_RULES block execution. SOFT_RULES score output. YAML-driven, zero tokens.

02

AST Verifier

Static analysis catches eval(), exec(), subprocess, hardcoded secrets before they run.

03

Meta-Supervisor

Weighted quality gate: Q(0.4) + A(0.25) + C(0.2) + F(0.15). Deterministic scoring.

04

Adversarial

Red Team attacks, Guardian defends, Arbiter decides. Prompt injection detection built in.

05

Memory Governance

Bi-temporal versioning. Constitutional decay. GovernedMemoryStore prevents drift.

06

Z3 Formal Proofs

SMT solver verifies invariants. Proof hash on-chain. VERIFIED or BLOCKED — no grey area.

07

Oracle Attestation

Merkle root → Avalanche C-Chain. Immutable. Verifiable. Forever.
