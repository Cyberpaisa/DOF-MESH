## PSYCHOLOGICAL FRAMEWORK APPLIED

| Technique | Where Used |
|---|---|
| **Loss Aversion** (people fear losing more than gaining) | Hook — "you already lost" framing |
| **Visceral Hook** — specific stat in 7s | Scene 1: "$2 trillion. Zero proof." |
| **Contrast Principle** | "Every framework verifies what happened. DOF verifies what's about to happen." |
| **Rule of 3** | Problem: 3 bullets. Proof: 3 layers. CTA: 3 asks. |
| **Social Proof** | 147 on-chain proofs, 8 chains, 4,308 tests — specific = credible |
| **Specificity = Credibility** | "7.3ms", "TRACER 0.712", "145 confirmed blocks" |
| **FOMO / Urgency** | "Regulations are coming. The infrastructure doesn't exist yet." |
| **YOU language** | Judges feel personally involved: "You wouldn't trust a bridge…" |
| **Sensory/Imaginative** | "Imagine an agent draining your DAO treasury at 3am…" |
| **Anchoring** | Big number first ($2T) → solution seems small and achievable |
| **The Contrast CTA** | "Math earns privilege. That's new." |

## DESIGN RULES (enforced in Remotion)

| Rule | Value |
|---|---|
| Background | `#0d0d0d` always |
| Primary text | `#FFFFFF` — never grey |
| Success/verified | `#00CC55` — max 20% frame |
| Accent | `#3B82F6` (Conflux blue) |
| Warning/problem | `#EF4444` (red) — problems only |
| Font | IBM Plex Mono — weight 600+ minimum |
| Title size | 96px (hook), 72px (section heads) |
| Body size | 40px minimum |
| Image overlay | `rgba(0,0,0,0.55)` strip — text on TOP of strip |
| Text over image | Bottom or right strip only — never over main content |
| Transitions | Fade 15 frames (0.5s) — no wipes or slides |
