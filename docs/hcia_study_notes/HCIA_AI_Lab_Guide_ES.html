1.2.2 Implementación de Regresión Lineal (Experimento Expandido)

Este experimento usa el conjunto de datos lr2_data.txt, que contiene datos simulados de área y precio de casas. Para obtener el conjunto de datos, consulta la Guía de Configuración del Entorno de Laboratorio.

Paso 1   Importar dependencias.
Entrada:
import numpy as np import matplotlib.pyplot as plt
Paso 2   Definir la función para calcular gradientes.
Entrada:
def generate_gradient(X, theta, y): sample_count = X.shape[0] # Calcular el gradiente basado en la matriz 1/m Σ(((h(x^i)-y^i)) x_j^i) return (1./sample_count) * X.T.dot(X.dot(theta) - y)
Paso 3   Definir la función para leer conjuntos de datos.
Entrada:
def get_training_data(file_path): orig_data = np.loadtxt(file_path, skiprows=1) # Ignorar el título en la primera fila cols = orig_data.shape[1] return (orig_data, orig_data[:, :cols - 1], orig_data[:, cols-1:])
Paso 4   Definir la función para inicializar parámetros.
Entrada:
# Inicializar el arreglo θ def init_theta(feature_count): return np.ones(feature_count).reshape(feature_count, 1)
Paso 5   Definir la función para implementar el descenso de gradiente.
Entrada:
def gradient_descending(X, y, theta, alpha): Jthetas = [] # Registrar la tendencia de cambio de la función de costo J(θ) # Calcular la función de pérdida: (y^i - h(x^i))^2 Jtheta = (X.dot(theta) - y).T.dot(X.dot(theta) - y) index = 0 gradient = generate_gradient(X, theta, y) # Calcular el gradiente while not np.all(np.absolute(gradient) <= 1e-5): # Terminar cuando el gradiente sea < 0.00001 theta = theta - alpha * gradient gradient = generate_gradient(X, theta, y) # Calcular la función de pérdida: (y^i - h(x^i))^2 Jtheta = (X.dot(theta) - y).T.dot(X.dot(theta) - y) if (index + 1) % 10 == 0: Jthetas.append((index, Jtheta[0][0])) # Registrar el resultado cada 10 cálculos index += 1 return theta, Jthetas
Paso 6   Definir la función para visualizar la curva de cambio de la función de pérdida.
Entrada:
# Graficar la curva de cambio de la función de pérdida def showJTheta(diff_value): p_x = [] p_y = [] for (index, sum) in diff_value: p_x.append(index) p_y.append(sum) plt.plot(p_x, p_y, color='b') plt.xlabel('pasos') plt.ylabel('función de pérdida') plt.title('curva paso - función de pérdida') plt.show()
Paso 7   Definir la función para visualizar puntos de datos y la curva ajustada.
Entrada:
# Graficar los puntos de datos reales y la curva ajustada def showlinercurve(theta, sample_training_set): x, y = sample_training_set[:, 1], sample_training_set[:, 2] z = theta[0] + theta[1] * x plt.scatter(x, y, color='b', marker='x', label="datos de muestra") plt.plot(x, z, 'r', color='r', label="curva de regresión") plt.xlabel('x') plt.ylabel('y') plt.title('curva de regresión lineal') plt.legend() plt.show()
Paso 8   Graficar los resultados finales.
Entrada:
# Leer el conjunto de datos training_data_include_y, training_x, y = get_training_data("./ML/02/lr2_data.txt") # Obtener el número de muestras y características sample_count, feature_count = training_x.shape # Definir el paso de aprendizaje α alpha = 0.01 # Inicializar θ theta = init_theta(feature_count) # Obtener el parámetro final θ y el costo result_theta, Jthetas = gradient_descending(training_x, y, theta, alpha) # Mostrar el parámetro print("w:{}".format(result_theta[0][0]), "b:{}".format(result_theta[1][0])) showJTheta(Jthetas) showlinercurve(result_theta, training_data_include_y)
Salida (ignorar advertencias):
w:3.0076279423997594 b:1.668677412281192
[Gráfico 1: Curva paso–función de pérdida: baja rápidamente de ~800 a ~0 en los primeros pasos] [Gráfico 2: Curva de regresión lineal ajustada sobre los datos de muestra]

1.2.4 Árbol de Decisión

Este experimento usa el conjunto de datos tennis.txt, que contiene 14 muestras. Cada muestra contiene características relacionadas con el clima e indica si es adecuado para jugar tenis.

Paso 1   Importar dependencias.
Entrada:
import pandas as pd import numpy as np from sklearn import tree import pydotplus
Paso 2   Definir la función para generar un árbol de decisión.
Entrada:
# Generar un árbol de decisión def createTree(trainingData): data = trainingData.iloc[:, :-1] # Matriz de características labels = trainingData.iloc[:, -1] # Etiquetas trainedTree = tree.DecisionTreeClassifier(criterion="entropy") # Clasificador de árbol de decisión trainedTree.fit(data, labels) # Entrenar el modelo return trainedTree
Paso 3   Definir la función para guardar el diagrama del árbol generado.
Entrada:
def showtree2pdf(trainedTree, finename): dot_data = tree.export_graphviz(trainedTree, out_file=None) # Exportar en formato Graphviz graph = pydotplus.graph_from_dot_data(dot_data) graph.write_pdf(finename) # Guardar el diagrama del árbol en formato PDF
Paso 4   Definir la función para generar datos vectorizados.

En la función, pd.Categorical(list).codes obtiene la lista de números de secuencia correspondientes a los datos originales, para convertir la información categórica en información numérica.

Entrada:
def data2vectoc(data): names = data.columns[:-1] for i in names: col = pd.Categorical(data[i]) data[i] = col.codes return data
Paso 5   Invocar la función para predicción.
Entrada:
data = pd.read_table("./ML/tennis.txt", header=None, sep='\t') # Leer datos de entrenamiento trainingvec = data2vectoc(data) # Vectorizar datos decisionTree = createTree(trainingvec) # Crear árbol de decisión showtree2pdf(decisionTree, "tennis.pdf") # Graficar el árbol de decisión

Se genera un diagrama del árbol de decisión llamado tennis.pdf en la máquina local.

Interpretación del árbol:
• X[2] = tercera variable de característica (humedad)
• X[0] = primera variable de característica (clima)
• X[3] = cuarta variable de característica (viento)
• entropy = valor de entropía del nodo
• samples = número de muestras (ej. 14 en el nodo raíz)
• value = número de muestras de cada tipo (ej. [5,9] = 5 "No" y 9 "Sí")
Predecir una nueva muestra. Entrada:
testVec = [0, 0, 1, 1] # Clima: soleado, temperatura: baja, humedad: alta, viento: fuerte print(decisionTree.predict(np.array(testVec).reshape(1, -1))) # Predecir
Salida:
['N']

1.2.5 Implementación del Algoritmo K-means

Paso 1   Importar dependencias.

make_blobs se usa para generar el conjunto de datos requerido para este experimento, matplotlib se usa para la visualización de datos, y KMeans se usa para ajustar los datos basándose en el algoritmo K-means.

Entrada:
from sklearn.datasets import make_blobs import matplotlib.pyplot as plt from sklearn.cluster import KMeans
Paso 2   Generar el conjunto de datos.

make_blobs() se usa para generar un conjunto de datos para una tarea de agrupamiento, devolviendo el conjunto de datos generado y las etiquetas. Nota: el algoritmo K-means a menudo se usa para procesar conjuntos de datos sin etiqueta en producción. El conjunto de datos generado contiene etiquetas solo para facilitar la comprensión.

Entrada:
X, y = make_blobs(n_samples=500, n_features=2, centers=4, random_state=1)

n_samples = número de muestras, n_features = número de características de cada muestra (dimensión de los datos), centers = número de categorías, random_state = semilla para el generador de números aleatorios (garantiza que los datos generados sean siempre iguales).

Entrada:
Print("Dimensión de X es {}".format(X.shape)) Print("Dimensión de y es {}".format(y.shape))
Salida:
Dimensión de X es (500, 2) Dimensión de y es (500,)

El conjunto de datos contiene 500 puntos de muestra, y cada punto contiene dos características.

Paso 3   Dibujar gráficos de dispersión.

Primero, dibujar directamente un gráfico de dispersión del conjunto de datos. Las etiquetas generadas durante la creación del conjunto de datos no se consideran por ahora. El gráfico contiene cuatro grupos distintos.

Entrada:
fig, ax1 = plt.subplots(1) ax1.scatter(X[:, 0], X[:, 1] , marker='o' # Establecer la forma del punto en círculo , s=8 # Establecer el tamaño del punto ) plt.show()
