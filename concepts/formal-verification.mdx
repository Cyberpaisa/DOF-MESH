## The Z3 Gate

**File:** `core/z3_gate.py` â€” neurosymbolic interface between LLM proposals and formal proofs.


| Verdict | Meaning |
|---------|---------|
| `APPROVED` | All constraints satisfied, proof complete |
| `REJECTED` | At least one constraint violated |
| `TIMEOUT` | Solver exceeded time budget (default: 5s) |
| `FALLBACK` | Z3 unavailable â€” rule-based governance only |
