DOF Sovereign Dashboard

Liquid Glass 2026 — Deterministic Observability Framework

Quick Start (Vite)

  • Install dependencies
npm install react react-dom recharts lucide-react
  • Start dev server
npx vite
  • Import the dashboard component in your React app
import Dashboard from './DOF_Dashboard.jsx'

With Existing React App

Copy DOF_Dashboard.jsx into your src/ directory and import it as a component.

// App.jsx import DOFDashboard from './DOF_Dashboard' export default function App() { return <DOFDashboard /> }

REST API Backend

The dashboard fetches data from the DOF REST API. Start the API server first:

python3 main.py # option 23: Start REST API Server

Default: http://localhost:8080/docs

DOF Dashboard requires a React environment (React 18+, Recharts, Lucide).
