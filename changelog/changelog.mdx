## v0.6.0 — April 2026

### Session 8 — Governance Hardening (Apr 12, 2026)

- **Test suite** grows to 4,272 tests, 0 failures
- **Self-correction loop** (`check_and_correct()`) — soft-rule violations repaired deterministically before blocking
- **Tool concurrency classification** — `CONCURRENT_SAFE_TOOLS`, `WRITE_TOOLS`, `classify_tool()` for Z3 bypass on read-only calls

- **Knowledge Pipeline** — 6-component end-to-end: YouTube → extractor → daemon → reporter → notifier → approver

- **knowledge_api.py** — HTTP server on port 19019 (`/health`, `/ingest`, `/pending`, `/approve/:rid`)
- **Auto-startup** — macOS LaunchAgent (`com.dof.mesh.plist`, RunAtLoad=true)
- **~/bin/dof** — Kill+restart+health check in one command
- 25 commits · 6 new core modules · 4 runtime bugs fixed
