- **v0.7.0 bump** вАФ `dof/__init__.py` + `pyproject.toml` вЖТ 0.7.0 ¬Ј 4,361 tests ¬Ј 145 modules
- **MANIFEST.in** вАФ PyPI exclusions: `_internal/`, `tests/`, `logs/`, `*.map` (prevents Claude CodeвАУstyle source leak)
- **DOFFeatureFlags** (`core/feature_flags.py`) вАФ runtime feature flag registry with YAML constitution loading, 3-layer priority (runtime override вЖТ YAML вЖТ defaults), 12 flags declared for v0.7.0 and v0.8.0 candidates

- **feature_flags section** in `dof.constitution.yml` вАФ all 12 flags with descriptions, Policy-as-Code for rollout control

### Session 8 вАФ Fase 2 & 3: Resilience + Streaming (Apr 12, 2026)
