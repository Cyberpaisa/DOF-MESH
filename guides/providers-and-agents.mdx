## Research Crew Pipeline

```
Pre-research (5 web searches)
  ↓ real data injected into context
t1: Researcher    (Groq)        → analyzes data
t2: Strategist v1 (Cerebras)   → drafts MVP plan
t3: QA            (Groq)        → verifies with additional searches
t4: Verifier      (Cerebras)   → scores + provides feedback
t5: Strategist v2 (Cerebras)   → produces FINAL improved plan
```
