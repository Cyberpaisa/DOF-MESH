## Overview

DOF-MESH integrates with Conflux Network at two levels:

- **Conflux eSpace** (production-ready) — `DOFProofRegistry` smart contract deployed on chain 71. Every governance proof is published as a keccak256 hash, permanently on-chain.
- **Conflux Core Space** (awareness layer) — native `cfx_*` RPC integration for network-level queries. Full Core Space transaction signing is on the roadmap.

## Why Conflux

Conflux's **Gas Sponsorship** (`SponsorWhitelistControl`) is architecturally necessary for autonomous agent infrastructure. Agents should not hold gas tokens to register compliance proofs — they should act. With Conflux's native sponsorship, proof registration costs the agent nothing.

No other EVM chain has this natively.

## eSpace Integration — ConfluxGateway

```python
from core.adapters.conflux_gateway import ConfluxGateway

# Connect to testnet
gw = ConfluxGateway(use_testnet=True, dry_run=False)
print(gw.w3.eth.chain_id)  # 71

# Gas Sponsorship contract
sponsor = gw.get_sponsor_contract()
# → SponsorWhitelistControl at 0x0888000000000000000000000000000000000001

# Publish a proof
from core.chain_adapter import DOFChainAdapter
adapter = DOFChainAdapter.from_chain_name("conflux_testnet")
result = adapter.publish_attestation(
    proof_hash="0x...",
    agent_id=1687,
    metadata="dof-v0.6.0 z3=4/4 tracer=0.91"
)
# → {"tx_hash": "0x...", "chain_id": 71, "gas_used": 373421}
```

## Core Space Integration — ConfluxCoreGateway

Core Space uses native `cfx_*` RPC — distinct from EVM/eSpace.

```python
from core.adapters.conflux_core_gateway import ConfluxCoreGateway

# Connect to Core Space testnet
core_gw = ConfluxCoreGateway(use_testnet=True, dry_run=True)

# Query network state
epoch = core_gw.get_epoch_number()    # current epoch
price = core_gw.get_gas_price()       # gas price in Drip
status = core_gw.get_status()         # chain_id, network_id, epoch

print(core_gw.describe())
# {
#   "type": "conflux_core_space",
#   "rpc_url": "https://test.confluxrpc.com",
#   "rpc_protocol": "cfx_*",
#   "address_format": "cfx: (base32)",
#   ...
# }
```

| Feature | eSpace | Core Space |
|---------|--------|------------|
| RPC | `eth_*` (EVM) | `cfx_*` (native) |
| Addresses | `0x...` | `cfx:...` (base32) |
| DOFProofRegistry | ✅ deployed | Roadmap |
| Web3.py | ✅ | ❌ (cfx_* raw) |
| Gas Sponsorship | ✅ SponsorWhitelistControl | ✅ Storage Collateral |
| DOF-MESH status | Production | RPC-aware |
